# Card

`<cds-aichat-card>` is an extension of the Carbon Design Systemâ€™s `<cds-tile>` component, designed specifically for AI Chat applications.

## Key Features

- **Rounded corners**: Makes use of rounded modifier scss utility. and can be controlled via `--cds-aichat-rounded-modifier-radius` CSS variable.
- **OOTB sub-components**: OOTB components `<cds-aichat-card-footer>` and `<cds-aichat-card-steps>` provide consistent styling and can be placed in the provided slots. You can also plug in your own components into these slots.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import "@carbon/ai-chat-components/es/components/card/index.js";
```

We provide a lit-react wrapper for this web component to consume in a react application and can be imported as below

```javascript
import ChatButton from "@carbon/ai-chat-components/es/react/chat-button.js";
```
