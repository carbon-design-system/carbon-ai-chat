## Overview

Use `<ErrorMessageSVG>` to display an illustrated error state SVG in React when a catastrophic or unrecoverable error has occurred. Supports both `light` and `dark` theme variants.

### Getting started

Here's a quick example to get you started.

```jsx
import ErrorMessageSVG from "@carbon/ai-chat-components/es/react/error-message-svg.js";

function ErrorState() {
  return <ErrorMessageSVG theme="light" />;
}

function ErrorStateDark() {
  return <ErrorMessageSVG theme="dark" />;
}
```

## Component API

<ArgTypes of={ErrorMessageSVGReactStories.Default} />

#### Props

- `theme`: Controls the visual theme of the SVG. Accepts `"light"` (default) or `"dark"`.
