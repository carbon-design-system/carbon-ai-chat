## Overview

Use `cds-aichat-error-message-svg` to display an illustrated error state SVG when a catastrophic or unrecoverable error has occurred. Supports both `light` and `dark` theme variants.

### Getting started

Here's a quick example to get you started.

```html
<cds-aichat-error-message-svg theme="light"></cds-aichat-error-message-svg>
```

```html
<cds-aichat-error-message-svg theme="dark"></cds-aichat-error-message-svg>
```
