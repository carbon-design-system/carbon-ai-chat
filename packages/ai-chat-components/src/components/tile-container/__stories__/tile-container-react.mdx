# AI Chat Tile Container

The `<TileContainer>` wraps the Carbon Design System's [Tile](https://web-components.carbondesignsystem.com/?path=/docs/components-tile--overview) component to apply AI Chat styles.
if using a footer, `cds-aichat-tile-container-footer` class name needs to be added to the footer element.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import TileContainer from "@carbon/ai-chat-components/es/react/tile-container.js";
```

<Markdown>{`${cdnJs({ components: ["tile-container"] })}`}</Markdown>
<Markdown>{`${cdnCss()}`}</Markdown>

### JSX

Wrap any carbon `<Tile>`, `<ClickableTile>` with `<TileContainer>` to get the styles.
The div containing the `cds-aichat-tile-container-footer` will have all the footer styles.

```jsx
import { TileContainer, Tile } from "@carbon/ai-chat-components";

<TileContainer>
  <Tile>
    <h5 className="heading-01 margin-bottom-04">
      AI Chat Tile styling wrapper
    </h5>
    <p className="body-01">
      The Carbon Design System provides a comprehensive library of
      components, tokens, and guidelines. We need to implement the new AI
      Chat component following Carbon's design principles and accessibility
      standards.
    </p>
  </Tile>
</TileContainer>
```
